# Contributing to pqc-5g6g-hybrid

Thanks for interest! This repo supports quantum-safe telecom research.

## How to Contribute
1. Fork repo.
2. Branch: `git checkout -b feat/add-falcon-benchmark`
3. Commit: `git commit -m "Add Falcon sim to main.py"`
4. PR: Describe changes, link to paper section.

## Guidelines
- **Experiments:** Add algos to main.py; update timings.
- **References:** Unique BibTeX; vary sources (IEEE/ACM/arXiv).
- **No Figures:** Text/tables only.
- Issues: Bug reports, new models.

Code of Conduct: Be respectful.
